# MoSCoW Priority Matrix — ChinaScape

This document captures feature priorities using the MoSCoW method (Must / Should / Could / Won't). Use this to guide MVP scope and early releases.

## Must Have (Critical)
- English Search Functionality — allow users to search attractions in English by name, city, or category.
- Chinese/English Address Flashcards — addresses in Chinese characters for showing to taxi drivers.
- Core Attraction Database —  attraction records with name, location, opening hours, and short description.
- Passport/ID Requirements info — clear "Before You Go" checklists per attraction (ID, booking windows, special rules).
- Suggested Itinerary Templates — starter 1-day and 2-day templates to help last-minute planners.

## Should Have (Important)
High value to users; app can function without them but they improve usability and trust.
- Cultural Explanations — etiquette, historical context, and short cultural notes for each attraction.
- Multimedia Content — photos and short videos on attraction pages.
- Ticketing Links — links or instructions to official booking pages (English versions where available).
- "Must-Try" Local Food section — recommended local dishes and nearby restaurants.
- Integrated Map Pins — map visualization with pins for attractions and itinerary stops.
- Offline Mode — allow users to download saved spots and itineraries for offline use.

## Could Have (Desirable)
Nice-to-have features to delight users; app can function without them.
- User Accounts — signup, profile, and sync across devices.
- Cost Estimator — estimated costs per itinerary (tickets, transport, meals).
- User Review System — allow verified international visitors to leave reviews and tips.
- Public Transport "How-to" videos — short guides explaining local transit use.

## Won't Have (Out of Scope)
Planned out-of-scope items for the near term.
- Push Notifications
- In-app Ticket Purchasing
- Live Hotel Booking
- Social Media Feed
