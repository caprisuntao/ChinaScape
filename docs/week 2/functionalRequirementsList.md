# Requirements Specification - ChinaScape

## Functional Requirements (FR)

| ID | Requirement | Description |
|---|---|---|
| FR1 | Attraction Search | Search by name, city, or category (Nature, History, Food). |
| FR2 | Itinerary Builder | Tool to drag-and-drop attractions into a daily schedule. |
| FR3 | Cultural Insights | Dedicated sections for "Etiquette," "Historical Context," and "Must-Try Food." |
| FR4 | User Profiles | Save favorite spots and personalized itineraries. |
| FR5 | Multi-language Support | Primary UI in English, with Chinese "Flashcards" for showing to taxi drivers. |
| FR6 | Multimedia Content | Each attraction page includes photos or short video clips, and location. |
| FR7 | Offline Mode | Allow users to download itineraries for offline use. |
| FR8 | Ticketing Links | Integrate or link to official ticket booking pages where available. |
| FR9 | Cost Estimator | Provide estimated costs per itinerary (tickets, transport). |

## Non-Functional Requirements (NFR)

| ID | Requirement | Description |
|---|---|---|
| NFR1 | Performance | Pages must load in under 5 seconds. |
| NFR2 | Localization | Dates and times must follow international formats (DD/MM/YYYY). |
